### Signal Logic

strictLongSignal  = bullBkActive and recentBullBos and inDiscount and bullConfirm and not macroIsBearish
strictShortSignal = bearBkActive and recentBearBos and inPremium  and bearConfirm and not macroIsBullish

### Macro Filter (decay-based)
```pine
var int bearBosCount = 0
var int bullBosCount = 0
if bearBosEvt
    bearBosCount += 1
    bullBosCount := math.max(0, bullBosCount - 1)
if bullBosEvt
    bullBosCount += 1
    bearBosCount := math.max(0, bearBosCount - 1)
macroIsBearish = bearBosCount >= 2
macroIsBullish = bullBosCount >= 2
```

### BK Labels + Lines — gated by macro
```pine
plotshape(bullBkTrig and not macroIsBearish, ...)
plotshape(bearBkTrig and not macroIsBullish, ...)
if bullBkTrig and not macroIsBearish  // line drawing
if bearBkTrig and not macroIsBullish  // line drawing
```

### OB Visualization — array-based
- Bull OBs: `line.new(bar_index - 1, dBodyMax, bar_index, dBodyMax)` — blue, persist until `close < body bottom`
- Bear OBs: same pattern — orange, persist until `close > body top`
- All valid OBs shown simultaneously (no cap)

### FVG Visualization — array-based
- Bull FVGs: blue boxes, `box.delete()` at midpoint violation
- Bear FVGs: red boxes, `box.delete()` at midpoint violation
- No grey fade — immediate delete
